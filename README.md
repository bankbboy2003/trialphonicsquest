## 🌟 About

PhonicsQuest is a single-file HTML5 game where students explore a space galaxy, complete English phonics missions on each planet, and earn XP, levels and badges. Built for Thai EFL learners at below-A1 to A1 level with full bilingual (English + Thai) support throughout.

**Live game:** https://bankbboy2003.github.io/trialphonicsquest/

---

## 🎮 Features

- 🌍 **8 Planets** across 2 galaxies (Short Vowels + Long Vowels)
- 📚 **7 Task types** per planet — from Listen & Choose to Speak & Record
- 🌟 **Planet Challenge** — creative CA tasks on every planet (no hearts lost)
- 🎤 **Real speech assessment** via Web Speech API
- 🔥 **Streak counter**, XP bar, Level system, Badges
- 🇹🇭 **Fully bilingual** Thai (L1) + English (L2) throughout all UI
- 📊 **Live teacher dashboard** with Firebase real-time scores
- 🎯 **Second chance** retry system — hint before losing a heart
- 📱 **Mobile responsive** — works on phone, tablet, laptop
